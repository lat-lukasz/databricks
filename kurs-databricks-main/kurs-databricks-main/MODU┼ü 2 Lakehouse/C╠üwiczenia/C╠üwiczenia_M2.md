# Ćwiczenia Moduł 2

1. Stwórz wymagane obiekty w Databricks Unity Catalog (Catalogi, Volumes, Schema)
2. Załaduj dane do Volume data, na początek wystarczy kilka plików z folderu Dane (repo).
3. Uruchom kod w każdym z notatników (bronze, silver, gold)
4. Po załadowaniu danych zrób count na wszystkich tabelach
5. Dodaj dolejną porcję danych i uruchom cały proces jeszcze raz. To pozwili Ci zasymulować inkrementalne ładowanie danych. 
    - Sprawdź count() na tabelach żęby potwierdzić czy mechanizm inkrementalnego ładowania działa poprawnie.
