# Ćwiczenia Moduł 1

1. Zaloguj się do **Databricks Free Edition**.  
2. Stwórz repozytorium w **GitHub** lub innym narzędziu do kontroli wersji.  
3. Załaduj pliki z kursu (**kurs-databricks.zip**).  
4. Podłącz repozytorium z Databricks Workspace **Databricks**.  
5. Przejdź przez wszystkie notatniki, uruchamiając poszczególne bloki kodu.  
6. Zapoznaj się z interfejsem **Databricks UI** i funkcjonalnością workspace.
